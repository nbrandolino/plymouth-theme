# Plymouth Theme
This repository holds my custom plymouth theme.

## Themes
1. **spinner-atomic**:
    - Custom theme based off the existing spinner theme. Changed background and logo to match grub theme.

## Install
To install, use the following commands:
```bash
cd ./plymouth-theme
```
```bash
make install
```
